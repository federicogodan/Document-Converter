unoconv instance  
================

The following steps describe the installation of one unoconv instance.

1 - Download unoconv tar unoconv-0.6.tar.gz

2 - Unpack tar file : tar -vxzf unoconv-0.6.tar.gz

3 - Move to directory created : cd unoconv-0.6/

4 - sudo make install
    In case of error, execute sudo install -p -m0755 unoconv /usr/bin/unoconv

5 - Check libreoffice version : /usr/bin/libreoffice --version
    It must be version 4.1

In case you have another version continue with steps 6 and 7 

6 - Uninstall libreoffice : sudo apt-get remove libreoffice 

7 - Update repository and install libreoffice 4.1
    sudo add-apt-repository ppa:libreoffice/libreoffice-4-1
    sudo apt-get update  
    sudo apt-get install libreoffice

8 - unoconv will be installed at : /usr/bin/unoconv  

s3cmd
=====

Follow these steps to install s3cmd to use s3 repository .

1- Install s3cmd : sudo apt-get install s3cmd.

2- Configure it : s3cmd --configure
There you have to put the API access-key and the API secret-key of your bucket.

3- Get file from a bucket : s3cmd get s3://bucket/file, where 'file' is the name of the file you want to download.

4- Put the file into an s3 bucket : s3cmd put file s3://bucket , where 'file' is the name of the file that you want to put 


Reference link: http://blog.celingest.com/2013/09/03/herramientas-gestion-s3/
